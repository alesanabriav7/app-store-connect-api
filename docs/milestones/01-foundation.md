# Milestone 1 - TypeScript Foundation

## Goal
Establish a clean, modular TypeScript project foundation replacing the initial Swift bootstrap.

## Decisions
- Use strict TypeScript with NodeNext modules.
- Use Vitest for fast unit tests.
- Separate code into Core, Domain, Data, and Feature modules.

## Validation
- `npm run typecheck`
- `npm run test`
- `npm run build`
